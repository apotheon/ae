Assert

The Assert module is simple a conatiner module for the core extension methods: assert, #expect, etc.

This module is included directory into the Object class.

assert(*args, &block) click to toggle source

Assert a operational relationship.

4.assert == 3

If only a single test argument is given then assert simple validates that it evalutate to true. An optional message argument can be given in this case which will be used instead of the deafult message.

assert(4==3, "not the same thing")

In block form, assert ensures the block evalutes truthfully, i.e. not as nil or false.

assert{ 4==3 }

def assert(*args, &block)
  Assertor.new(self, :backtrace=>caller).assert(*args, &block)
end

Alias for refute. Read it as “assert not”.

NOTE: This method would not be necessary if Ruby would allow +!=+ to be define as a method, or at least +!+ as a unary method. Looks like this is possible in Ruby 1.9, but we will wait until Ruby 1.9 is the norm.
